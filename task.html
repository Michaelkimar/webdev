Lists

Lists is a collection which is ordered and changeable using list methods

List Methods

Pop

shows the number/name occupying the entered index

Append

adds a number/name to the List

Sort

arranges the numbers in the list in ascending order

Count

counts the number of times a number/name appears in a list

Insert

inserts a number/name to a specific position

Remove

removes a number/name from a list
